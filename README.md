# 💱 Currency Converter App

A simple currency converter application built with React using exchange rate API.

## 🚀 Features
- Fetches exchange rate data from an API
- Currency conversion based on user input
- Simple and responsive interface
- Practice project for API integration in React

## 🛠️ Technologies
- React
- JavaScript
- CSS
- Fetch API / Axios

## ⚙️ Installation
```bash
npm install
npm run dev
